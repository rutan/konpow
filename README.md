# Konpow

*W.I.P*

Konpow is a gem that provides helpers for external asset builders (ex. webpack).

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'konpow'
```

## Usage

```
bundle exec rails g konpow:config
```
